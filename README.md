## Requirements

- GNU Make >= 4.4
- LLVM Project >= 15.0.7
    - Clang
    - LLD
    - llvm-objcopy
- wget >= 1.21.3
- (optional) tmux >= 3.3a
- (optional) qemu-system-arm

You can get them on macOS with Homebrew:

```sh
brew install make llvm tmux qemu wget
```

## Build 

```
make
```

## Test With QEMU

```
make run
```
